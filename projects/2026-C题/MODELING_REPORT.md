# 建模报告

## 模型总览

| 问题 | 模型 | 算法 |
|------|------|------|
| Q1 社群发现 | Louvain 社区发现 | 模块度优化 |
| Q2 好友推荐 | 多维度加权相似度 | 6 维特征加权评分 |
| Q3 信息传播 | 独立级联模型 (ICM) | 蒙特卡洛模拟 30 次 |
| Q4 精准推送 | 贪心选择 + 社群感知 | 文化参与度 × 度因子排序 |

## 关键参数

### 转发概率公式
$$P = 0.4 \times \frac{参与度}{10} + 0.3 \times \frac{互动频率}{25} + 0.3 \times \frac{1}{1+\Delta t}$$

### 好友推荐权重
| 维度 | 权重 |
|------|------|
| 共同好友 Jaccard | 0.30 |
| 同年级 | 0.10 |
| 同专业类别 | 0.15 |
| 共同社团 Jaccard | 0.20 |
| 同活跃时段 | 0.10 |
| 行为余弦相似度 | 0.15 |

## 核心结论

1. 社交网络存在明显的社群结构（模块度 Q=0.2586），社群间无成员重叠
2. 好友推荐的关键是共同社团 + 同活跃时段 + 行为相似度
3. 科技类话题传播关键用户是高参与度 + 高互动频率的用户（S6）
4. 精准推送能达到 191 人传播范围
